# Khmer Digital Font Usage Guidelines

## Overview
Khmer Digital is an official typeface developed by the Ministry of Post and Telecommunications of Cambodia (MPTC).

## Supported Platforms
- Windows
- macOS
- Linux
- Microsoft 365 applications (Word, Excel, Teams)


## Font Weights

| Weight | Name       |
|--------|------------|
| 100    | Thin       |
| 200    | ExtraLight |
| 300    | Light      |
| 400    | Regular    |
| 500    | Medium     |
| 600    | SemiBold   |
| 700    | Bold       |
| 800    | ExtraBold  |
| 900    | Black      |

## Recommendation

- Use Regular (400) for body text
- Use Bold (700) for emphasis
- Use Medium (500–600) for UI
